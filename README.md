#  AlphaMatrix AI

AlphaMatrix AI is an enterprise-grade AI-powered investment research and portfolio management platform. It provides intelligent market analysis, AI-driven insights, portfolio tracking, and multi-agent financial research using Google's Gemini AI.

---

##  Features

-  AI Investment Research Assistant
-  Portfolio Management Dashboard
-  Live Market Overview
-  Watchlist Management
-  AI Financial Research
-  Multi-Agent AI Chat
-  Secure Authentication
-  Modern Responsive UI
-  Fast Performance with Vite + React

---

## 🛠️ Tech Stack

### Frontend
- React 19

- Vite
- Tailwind CSS
- React Router
- Zustand

- Lucide React

### Backend
- Node.js
- Express.js
- TypeScript

##  Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/alphamatrix-ai.git
```

Go into the project folder

```bash
cd alphamatrix-ai
```

Install dependencies

```bash
npm install
```

---

##  Environment Variables

Create a `.env` file in the project root.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

##  Run Development Server

```bash
npm run dev
```

Server will run at

```
http://localhost:3000
```

---

##  Build Project

```bash
npm run build
```

---

##  Production

```bash
npm start
```
